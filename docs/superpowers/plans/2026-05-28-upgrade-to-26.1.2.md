# AdvancedAFK 26.1.2 Sürüm Geçişi Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Upgrade AdvancedAFK plugin to Minecraft/Paper version 26.1.2, targeting Java 25.

**Architecture:** We will modify the Maven pom.xml configuration to bump the Java version to 25 and the Paper API version to [26.1.2.build,). We will also update the plugin.yml configuration to register compatibility with the 26.1 API, and compile the codebase to verify and patch any compatibility errors.

**Tech Stack:** Java 25, Paper API 26.1.2, Maven, Bukkit/Spigot Plugin Framework

**Files:**
- Modify: `pom.xml:15-18`
- Modify: `pom.xml:32-38`

- [ ] **Step 1: Update Java version target to 25**

Update `<java.version>` property in `pom.xml`:
```xml
    <properties>
        <java.version>25</java.version>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
    </properties>
```

- [ ] **Step 2: Update paper-api dependency version to 26.1.2**

Update the `paper-api` dependency in `pom.xml` to use the `[26.1.2.build,)` range:
```xml

            <groupId>io.papermc.paper</groupId>
            <artifactId>paper-api</artifactId>
            <version>[26.1.2.build,)</version>
            <scope>provided</scope>
        </dependency>
```

- [ ] **Step 3: Verify pom.xml syntax and execute test compile**
